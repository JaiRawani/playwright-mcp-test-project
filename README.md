# Playwright MCP Test Project 🚀

This repository contains a sample Playwright project created to explore **Playwright MCP (Model Context Protocol)** and AI-assisted test generation.

The objective of this project is to understand how an AI agent can generate, execute, and iterate Playwright tests automatically using defined scenarios.

---

## 📌 Project Objective

This project was created to:

- Learn **Playwright automation testing**
- Explore **MCP based AI driven test generation**
- Practice **TypeScript based Playwright testing**
- Understand how AI tools like **GitHub Copilot** assist in writing automation tests

---

## 🧪 Test Scenario

The automated test covers the following scenario:

1. Navigate to  
   https://automationexercise.com/

2. Search for  
   **"T-shirts"**

3. Verify the product  
   **"Faded Short Sleeve T-shirts"** appears in the results list.

## ⚙️ Installation

Clone the repository:


git clone https://github.com/YOUR_GITHUB_USERNAME/playwright-mcp-test-project.git


Navigate to the project directory:


cd playwright-mcp-test-project


Install dependencies:


npm install


Install Playwright browsers:


npx playwright install


---

## ▶️ Running Tests

Run all tests:


npx playwright test


Run tests in headed mode:


npx playwright test --headed


Run Playwright UI mode:


npx playwright test --ui


---

## 📊 View Test Report

To view the HTML test report:


npx playwright show-report
